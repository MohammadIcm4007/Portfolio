## 🚀 شروع به کار  

### 1️⃣ کلون کردن مخزن  
```bash  
git clone https://github.com/your-username/react-login.git  
cd react-login  
```  

### 2️⃣ نصب وابستگی‌ها  
```bash  
npm install  
```  

### 3️⃣ اجرای برنامه  
```bash  
npm start  
```  
برنامه در آدرس `http://localhost:5173` در دسترس خواهد بود.
